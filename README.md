# Building-Table-with-Python
"Создание таблиц в Python c использованием бибилиотеки PrettyTable" Pretty Table  в Python позволяет создавать таблицы ASCII для отображения данных в терминале. 
Она поддерживает работу не только с ASCII-таблицами, но и с HTML, CSV и SQL-форматами данных. 

#Необходимые бибилиотеки для реализации проекта:
                                          1) prettytable
